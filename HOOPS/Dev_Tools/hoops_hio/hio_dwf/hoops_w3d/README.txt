This is HOOPS/W3D module. It is the source code provided by Autodesk (refered there as W3DHOOPS). It is essentially Autodesk's version of HOOPS/Stream. It is used to read-write 3D portion (W3D) of DWF files.
HOOPS/W3D is used by HOOPS/DWF module to export and import 3D DWF files.
